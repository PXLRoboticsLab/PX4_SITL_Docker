# SITL gazebo docker
* Firmware is located at ~/Projects/Firmware
* catkin is located at ~/Projects/catkin

## Clone repo
```
git clone https://github.com/PXLRoboticsLab/PX4_SITL_Docker.git
```

## Build the docker

```
cd PX4_SITL_Docker
bash build_image.sh
```

## Start the docker

```
cd PX4_SITL_Docker
bash start_container.sh
```
